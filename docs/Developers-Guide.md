# Developers Guide

## Run locally

### Backend
```bash
WEBINPUT_CSV_SESSION_CONFIG=/path/to/config.file hug -f intelmq_webinput_csv/serve.py -p 8002
```
Intelmq needs to be installed as it is a dependency.

### Frontend

```bash
yarn run dev
```
